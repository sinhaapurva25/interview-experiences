Run the project in the following way:

1. Download the zipped file and unzip it.
2. Go inside directory "...ImageProcessingChallenge\output" and delete the all the image files in it.
3. Open "...ImageProcessingChallenge\src\imageprocessing.sln" with Microsoft Visual Studio 2019.
4. From the Menu Bar of the Microsoft Visual Studio 2019, open Tools > Command Line > Developer Command Line
5. In the Command Line, go to the main assignment directory.
	Write cd <path to the main assignment directory> and hit enter.
6. Type the nmake command, that is, type "nmake /f NMakefile" and hit enter.
7. Check the output folder. It should have images in it.
